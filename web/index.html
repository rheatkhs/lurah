Platform Overview

Lurah is a cross-platform static code analysis tool written in Go, specifically engineered for high-security Laravel ecosystems. It streamlines the audit process by mapping code vulnerabilities directly to Indonesian SPBE Standards.

CLI Command Reference

# Audit the current directory

lurah scan

# Launch the web dashboard on port 9999

lurah web --port 9999

# Watch for file changes and re-scan

lurah watch --path ./app

Customization

Define custom security rules in your .lurah.yaml file to target proprietary patterns. 

custom_rules:
  - name: "no-dd"
    pattern: "\\bdd\\("
    severity: "HIGH"
    message: "Debug function found in production code."
